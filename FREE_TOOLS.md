# Free Tools — SaaS Lead Generator v2.0

> Zero paid APIs. Zero config. Works out of the box.

## Tools Used

| Tool | Purpose | Cost |
|------|---------|------|
| DuckDuckGo API | Web search | Free, no key |
| OpenCorporates API | Company registry search | Free tier |
| Clearbit Logo API | Company logos | Free, no key |
| Node.js DNS | MX record validation | Built-in |
| Cheerio | Email scraping from websites | npm package |
| LinkedIn (manual URLs) | Decision maker search | Free account |
| Clutch | Agency directory | Free browsing |
| GoodFirms | Agency & firm listings | Free browsing |
| Google Search URLs | Agency + small team queries | Free |

## Quick Start

```bash
npm install
npm run dev
# Open http://localhost:3000 — no .env needed!
```
